# IPTV Pro 📺

تطبيق IPTV عربي يحتوي على 1700+ قناة — المغرب، beIN، TOD، شاهد، الأخبار، وغيرها.

## بناء APK على GitHub

### الخطوة 1 — رفع المشروع على GitHub
1. أنشئ Repo جديد على GitHub
2. ارفع محتوى هذا المجلد كاملاً

### الخطوة 2 — احصل على EXPO_TOKEN
1. اذهب إلى https://expo.dev وأنشئ حساباً مجانياً
2. اذهب إلى **Settings → Access Tokens**
3. اضغط **Create Token** وانسخه

### الخطوة 3 — أضف السر في GitHub
1. في الـ Repo اذهب إلى **Settings → Secrets → Actions**
2. اضغط **New repository secret**
3. الاسم: `EXPO_TOKEN` والقيمة: الـ Token الذي نسخته

### الخطوة 4 — شغّل البناء
1. اذهب إلى تبويب **Actions** في الـ Repo
2. اختر **EAS Build APK**
3. اضغط **Run workflow**
4. انتظر ~10 دقائق
5. حمّل الـ APK من https://expo.dev/accounts أو من تبويب **Artifacts**

## تثبيت APK على الهاتف
1. أرسل ملف APK لهاتفك
2. افتح الملف → اقبل "تثبيت من مصادر غير معروفة"
3. ✅ التطبيق مثبّت دائماً على هاتفك
